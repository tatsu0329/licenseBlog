# AdSenseポリシー準拠ガイドライン

このドキュメントでは、Google AdSenseのポリシーに準拠するためのガイドラインをまとめています。

参考: [AdSense Program policies](https://support.google.com/adsense/answer/48182)

## 重要なポリシー

### 1. 無効なクリック・インプレッションの防止

**禁止事項:**
- 自分の広告をクリックすること（どんな理由でも禁止）
- 人工的なクリックやインプレッションを生成すること
- 自動クリックツールやロボットを使用すること

**対策:**
- 広告を表示する場合は、自分でクリックしない
- 家族や友人にもクリックを依頼しない
- 自動化ツールやサービスを使用しない

### 2. クリックやビューを促すことの禁止

**禁止事項:**
- 「広告をクリックしてください」
- 「サポートしてください」
- 「応援してください」
- 「これらのリンクを訪問してください」
- 広告の近くに矢印や画像を配置して注意を引くこと
- 誤解を招く画像を広告の隣に配置すること

**対策:**
- 広告の近くに、クリックを促すような文言を配置しない
- 広告に矢印や誘導するようなアイコンを配置しない
- 広告が自然に見えるように配置する

### 3. 広告のラベル

**許可されるラベル:**
- 「広告」
- 「Advertisements」
- 「Sponsored Links」

**禁止されるラベル:**
- 「お気に入りサイト」
- 「今日のお得情報」
- 「推奨リンク」
- 「人気サイト」
- 「必見リンク」

**実装:**
- `AdSense`コンポーネントでは、デフォルトで「広告」ラベルを表示
- ラベルは小さい文字で、広告の上に配置

### 4. 広告の配置

**禁止される配置:**
- ポップアップやポップアンダー内
- メール内
- ソフトウェアアプリケーション内（AdMob除く）
- 非コンテンツページ（AdSense for searchを除く）
- 広告専用ページ
- フレーム内のコンテンツ

**推奨される配置:**
- コンテンツページのヘッダー下
- コンテンツの中（適切な間隔を空ける）
- サイドバー
- フッター上

### 5. サイトの動作

**禁止事項:**
- ユーザーの設定を変更すること
- 不要なサイトにリダイレクトすること
- 自動ダウンロードを開始すること
- マルウェアを含むこと
- ポップアップやポップアンダーでナビゲーションを妨害すること

### 6. トラフィックソース

**禁止されるトラフィックソース:**
- 有料クリックプログラム（paid-to-click）
- 有料サーフィンプログラム（paid-to-surf）
- クリック交換プログラム
- スパムメール
- 不要な広告
- ツールバーなどのソフトウェアアプリケーション

## 実装上の注意点

### AdSenseコンポーネントの使用

```tsx
import AdSense from "@/components/AdSense";

// 基本的な使用例
<AdSense adSlot="1234567890" />

// ラベルをカスタマイズ（許可されるラベルのみ）
<AdSense adSlot="1234567890" label="広告" />
<AdSense adSlot="1234567890" label="Advertisements" />
<AdSense adSlot="1234567890" label="Sponsored Links" />
```

### 広告配置のベストプラクティス

1. **適切な間隔を空ける**
   - 広告とコンテンツの間に十分なスペースを確保
   - モバイルでは特に注意

2. **コンテンツと区別できるようにする**
   - 広告ラベルを必ず表示
   - スタイリングでコンテンツと明確に区別

3. **ユーザー体験を優先**
   - 広告が多すぎないようにする
   - コンテンツの読みやすさを損なわない

4. **モバイル最適化**
   - レスポンシブな広告を使用
   - モバイルでは広告数を減らすことを検討

## チェックリスト

広告を配置する前に、以下を確認してください:

- [ ] 広告ラベルが適切に表示されている（「広告」「Advertisements」「Sponsored Links」のいずれか）
- [ ] 広告の近くにクリックを促す文言がない
- [ ] 広告に矢印や誘導アイコンが配置されていない
- [ ] 広告がコンテンツと明確に区別できる
- [ ] ポップアップやポップアンダー内に広告がない
- [ ] メール内に広告がない
- [ ] 広告専用ページでない
- [ ] ユーザー体験を損なわない配置になっている

## 違反時の対応

AdSenseポリシーに違反した場合:
1. 広告の配信が停止される可能性がある
2. AdSenseアカウントが無効化される可能性がある
3. 収益が支払われない可能性がある

違反を防ぐために、定期的にこのガイドラインを確認し、サイトをチェックしてください。
