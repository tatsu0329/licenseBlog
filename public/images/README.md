# 画像ファイルの配置場所

## アプリ関連画像

- `/images/apps/` - アプリのアイコンやスクリーンショット
  - `auto-mechanic-1-icon.png` - 1級自動車整備士アプリのアイコン
  - `auto-mechanic-1-screenshot1.png` - スクリーンショット1
  - `auto-mechanic-1-screenshot2.png` - スクリーンショット2
  - `auto-mechanic-1-screenshot3.png` - スクリーンショット3

## アイコン

- `/images/icons/` - 機能アイコン
  - `book.svg` - 書籍アイコン
  - `explanation.svg` - 解説アイコン
  - `analysis.svg` - 分析アイコン
  - `progress.svg` - 進捗アイコン

## App Store / Google Play バッジ

- `/images/app-store-badge.svg` - App Store バッジ
- `/images/google-play-badge.svg` - Google Play バッジ

これらのバッジは、Apple と Google の公式サイトからダウンロードできます：
- App Store: https://developer.apple.com/app-store/marketing/guidelines/#section-badges
- Google Play: https://play.google.com/intl/en_us/badges/
